核心要义

  1. 长期持有是意向而非教条：买入那一刻是打算长期持有的，但机会成本永远在变动，不应教条式持有。
  2. 好公司不需要卖：真正的好公司，越持有价值越大，卖出反而会损失复利的力量。
  3. 烂生意长期持有是灾难：时间是好生意的朋友、烂生意的敌人。长期持有烂生意只会获得烂收益。
  4. 机会成本是唯一的衡量标准：如果能找到更好的公司，换仓没有什么不对。

经典案例

  • 网易：段永平十几年的长期持有获得了百倍回报，但不是因为「长期持有对」，而是因为网易是好公司。
  • GE：段永平在发现GE的问题后卖出，没有因为「长期持有」的教条而死守。当公司的生意模式出现问题时，卖出是正确的。

演变轨迹

  • 早期（2010-2012）：明确反对「长期持有=价值投资」的误解，提出「长期持有是意向」。
  • 中期（2013-2018）：持续强调好公司不需要卖，但也不会死守。以机会成本为衡量标准。
  • 近期（2019-2025）：2025年访谈中再次确认这一灵活立场，与市场上「死了都不卖」的教条式长期持有形成鲜明对比。

常见误解

  • ❌ "长期持有=价值投资" → 段永平认为这是对价值投资最大的误解。长期持有是结果，不是目的。
  • ❌ "买了就必须长期持有" → 如果有更好的机会就应该换，机会成本是唯一标准。
  • ❌ "长期持有=不关心价格" → 买入价格仍然重要，好公司也需要便宜的价格。
