段永平怎么说

犯错与纠错是段永平思想体系中的核心模块，与"做对的事情，把事情做对"紧密关联。他区分了两类错误：做错的事（不可接受、必须立即停止）和把事情做错（学习过程、可以接受）。好的企业文化能在犯错后自我纠正让公司走回正道。Stop Doing List就是通过血的教训一条一条积累起来的。

核心要义

  1. 两类错误严格区分：做错的事（方向性错误）必须立即停止；把事情做错（执行性错误）是学习过程，可以接受
  2. 发现错误马上改：纠错的速度决定损失的大小——"你发现错了，一定要马上停"
  3. 好文化降低犯错代价：企业文化越好的公司，犯错后被纠正的概率越高，最终会走回正道
  4. Stop Doing List来自血的教训：不是靠聪明想出来的，是犯错后总结出来的——每一次大错对应一条Stop Doing
  5. 挣钱的投资也可能是错误：GE案例说明，一个投资即使最终赚了钱，如果决策逻辑是错的，它仍然是错误

经典案例

GE：挣了钱的错误

段永平坦诚："GE...虽然挣了钱你也不能否认它是个错误。回到今天我是不会买的。"——这是纠错思维最深刻的体现：不以结果论对错。即使最终赚了钱，当初的决策逻辑（未充分评估商业模式和企业文化）是错误的，就应该承认错误并从中学习。

乐视：错的事情任何价格卖都是对的

段永平回忆有人问他乐视三十几块怎么样，他的回答是"价钱很好啊，因为将来会是零的嘛，所以你现在什么价格卖都是好的。"——当你确认自己做了"错的事情"，不要纠结于成本价，任何价格退出都是正确的。

百度的做空错误

段永平早年做空百度的教训成为"不做空"原则的直接来源——"不做空。这方面我曾经犯过错误，尤其是不要做空长期的好公司，比如早年的百度。"一次错误换来一条终身遵守的原则。

Stop Doing List的形成过程

2018年斯坦福演讲中，段永平详细阐述了Stop Doing List的积累过程：每一条背后都有血的教训。不做代工、不借钱、不做空、不进入不熟悉的领域——每一条都是经历过错误后提炼出来的规则。

演变轨迹

  • 2010年前：早期创业和投资中积累了各种错误教训（百度做空、早期投资失误）
  • 2010-2013年：在网易博客中开始系统化讨论"做对的事情"与"把事情做对"的区别
  • 2013-2015年：GE案例的反思路——挣了钱但依然承认是错误，体现纠错思维的成熟
  • 2018年：斯坦福演讲系统阐述Stop Doing List的形成过程和"两类错误"的理论框架
  • 2019-2024年：在雪球上通过乐视等案例反复向网友阐述"发现错误马上改"的原则
  • 2025年：方三文对话中总结"企业文化在纠错中的作用"和"30年犯错比别人少"的长期验证

常见误解

误解一："犯错=失败"

段永平认为犯执行性错误（把事情做错）是成长的必经之路——“犯错误的概率大家其实都差不多的”。关键不是不犯错，而是不犯方向性错误（做错的事）和知错就改。

误解二："赚了钱就不是错误"

GE案例是段永平最有力的反驳——即使赚了钱，如果决策逻辑是错误的，它就是错误。不以结果论对错，是纠错思维的最高境界。

误解三："纠错=否定自己"

段永平的态度极其务实："你发现错了，一定要马上停。"纠错不是面子问题，是生存问题。能快速纠错恰恰是最强的能力。

误解四："Stop Doing List一次制定，终身不变"

Stop Doing List是动态积累的过程——"我们也是要通过很多血的教训才知道，然后知道这个事不能做，我们以后就不做了。"每一条都是后验的，不是预设的。
