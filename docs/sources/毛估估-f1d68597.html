核心要义

  1. 模糊的正确优于精确的错误：与其用复杂公式算出一个「精确」的数字，不如用常识判断大方向。
  2. 5分钟原则：真正的便宜货应该5分钟就能判断。如果需要复杂的计算和假设，那可能就不够便宜。
  3. 定性为主、定量为辅：真正赚大钱靠的是定性分析（生意模式、企业文化），毛估估是对定性判断的粗略数值验证。
  4. 300斤胖子不需要秤：好机会明显到不需要精心计算，一眼就能看出便宜。

经典案例

  • 万科：市值100多亿时买入，段永平认为「这无论如何也不止」，随便给了个500亿估值。没有复杂计算，纯靠毛估估就判断出明显低估。
  • 网易：买网易时根本没认真「估值」过。当时网易现金多、游戏业务在增长、市值极低，就像300斤的胖子——不用秤也知道。
  • GE：认为值20块，15块开始买一点，10块以下重手。估值是模糊的区间判断，而非精确的数字。

演变轨迹

  • 早期（2010-2012）：在DCF文章中系统阐述毛估估概念，强调估值不需要公式。
  • 中期（2013-2018）：在《复习一下》中收集大量实战案例（万科、GE、网易），用经验说明毛估估的有效性。
  • 近期（2019-2025）：核心观点始终不变。2025年雪球讨论中仍频繁使用「毛估估」一词，强调5分钟看不懂就不值得深入。

常见误解

  • ❌ "毛估估=随便估" → 毛估估的前提是在能力圈内。不懂的公司再毛估估也没意义。
  • ❌ "毛估估就是不用分析" → 毛估估是定性分析后的粗略定量验证，不是偷懒的借口。
  • ❌ "毛估估不需要DCF" → 毛估估的本质就是用DCF思维方式做定性判断，不是否定DCF。
