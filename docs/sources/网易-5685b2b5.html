段永平怎么说

段永平最著名的投资案例——2001年底以1美元左右价格大量买入网易股票（均价约0.16美元），后来获利超100倍。投资逻辑核心：他懂游戏行业（做过小霸王），花三个月让团队每天玩游戏确认看懂了才重仓。从能力圈扩展到确定性判断的经典案例。

2001年底，他开始以1美元左右价格大量买进网易股票，并坚持持有到100美元。华尔街的华人投资圈因此而送他"段菲特"的称号。

投资时间线

  • 2001年底：开始以1美元左右价格大量买进网易股票
  • 2002年：斥资200万美元以均价0.16美元买入约5%的网易股票
  • 2003年：网易网游《大话西游》爆发，股价开始飙升
  • 2003-2004年：股价涨至40美元，开始减持大部分持仓（"卖股票和成本无关"）
  • ~2005-2006年：股价继续涨至70-80美元，仍持有部分仓位（澄清未全部卖出）
  • 2009年Q1：仍持有超400万股网易股票（当时股价约291美元）
  • 2025年：方三文对话中回顾，坦诚当时的"懂"是程度上的事

核心要义

  1. 能力圈扩展的方法论：不是凭空判断——花三个月让团队每天玩游戏，确认看懂了网游商业模式才重仓
  2. 危机中的机会：网易当时面临退市风险、财务危机、股价跌至0.13美元——极端低估创造了超额回报
  3. 卖出决策基于价值，而非成本："卖股票和成本无关"——40美元卖出大部分是基于价值判断，不是被0.16美元的成本锚定
  4. "懂"是程度问题：2025年坦诚反思——如果真正完全"懂"了，应该把整个网易买下来
  5. 管理层判断："丁磊是个很有悟性的人"——对人的判断也是投资的重要组成部分

经典案例

从0.16美元到100美元+

这是段永平投资生涯中最传奇的案例。2001年互联网泡沫破裂后，网易股价从15.5美元暴跌至0.13美元（面临退市风险）。段永平利用自己做过小霸王/步步高游戏业务的背景，判断网易的网游业务有强大的变现能力。他让团队花三个月深度体验网易游戏后才确认买入。最终这笔投资获得了超过100倍的回报。

卖出的理性决策

当股价涨到40美元时，段永平开始减持。这不是因为"赚够了"，而是重新评估后认为当前价格已经反映了价值。他说："买的时候认为至少值得价钱，到了以后应该很认真地再研究一下。"

演变轨迹

  • 2001-2002年：深入研究网易游戏业务，确认商业模式后重仓买入
  • 2003-2006年：网游《大话西游》《梦幻西游》大获成功，股价持续攀升
  • 2010年：在博客中反复引用网易案例解释"懂"和"坚持"的关系
  • 2010-2015年：将网易投资经验提炼为"能力圈""确定性""安全边际"等概念
  • 2025年：回顾时更加谦虚——承认当时的"懂"有运气成分

常见误解

误解一："段永平在70-80美元全卖了"

他澄清："其实当时70-80我也没卖，不知道为什么大家都说我卖了？"——他是在40美元左右减持了大部分，但并未全部卖光。

误解二："他完全看懂了网易"

2025年他坦诚反思："我要看懂了，我应该把整个网易买下来。对不对？"——"懂"从来不是100%。

误解三："这是一次赌博"

不是赌博。他花了三个月让团队每天玩游戏才确认决策，"不是凭空判断，是先扩展能力圈再行动。"
