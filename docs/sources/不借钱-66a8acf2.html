段永平怎么说

巴菲特原话是"Don't use margin!"，段永平简化为"不借钱"。核心是反对用融券方式借钱投资——市场大跌时会被强制平仓。巴菲特说"一生只需富一次"，借钱可能让你再也没有机会。段永平的公司步步高也基本不贷款。

核心要义

  1. 强制平仓是最大风险：借钱投资最致命的不是亏钱，而是在市场暴跌时被强制平仓——"没人强制平仓，就能真正做到忽略波动，只看分红"
  2. "一生只需富一次"：巴菲特的核心教诲——不要冒不应该冒的风险，不需要富第二次
  3. 了解投资 vs 借钱投资的悖论："如果你不了解投资的话，你不应该借钱。如果你了解投资的话，你不需要借钱。反正你早晚都会有钱的"
  4. 企业经营与投资一致：步步高/OPPO/vivo基本不贷款，段永平将同样的原则应用于个人投资
  5. 区分不同"借钱"：融券（做空）≠ margin（杠杆做多），但两者都应该避免

经典案例

NVR：从破产边缘到不借钱的典范

美国房地产公司NVR曾因太多负债而差一点破产，股票跌到过0.2美元/股。后来侥幸活过来后"就再也不借钱了"，几十年后股价涨到约700美元/股。段永平用这个案例说明"不借钱"是能活下来并活得好的关键。

董宝珍茅台爆仓

2013年白酒塑化剂危机+反腐期间，知名投资者董宝珍因使用杠杆重仓茅台而被强制平仓（爆仓），虽然茅台后来确实大幅回升，但董宝珍已无缘享受。这个案例成为段永平反复用来警示杠杆风险的经典例子。

巴菲特1998年佛罗里达演讲

巴菲特说："我只有1万块钱的时候都不借钱，不借钱不一样吗？我钱少的时候做投资也很开心。"——钱少不是借钱的借口。

演变轨迹

  • 2007年：波士堂访谈中已提到不借钱给朋友的原则
  • 2010年：博客明确三大原则含"不借钱"，并澄清来自巴菲特的"Don't use margin"
  • 2011年：解释巴菲特的"一生只需富一次"与不借钱的关联
  • 2013-2014年：用董宝珍茅台爆仓案例不断警示杠杆风险
  • 持续到2025年：雪球上仍在反复强调，频率极高

常见误解

误解一："杠杆就是借钱买股票"

段永平区分了不同形式：融券是抵押股票借钱、margin是以股票为抵押品借钱买更多股票。两者都属借钱范畴，但风险形式略有不同。

误解二："少量借钱没事"

"借钱是危险的，没人知道市场到底有多疯狂（向下或向上）。"——即使是"少量"，一旦遭遇极端行情也可能被强制平仓。

误解三："机会难得，借钱可以抓住机会"

"反正你借不借钱一生当中都会失去无穷机会的，但借钱可能会让你再也没机会了。"——机会错过了还有，本金没了就没机会了。
