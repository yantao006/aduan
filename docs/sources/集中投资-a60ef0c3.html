核心要义

  1. 巴菲特「20个打孔位」理念的实践者：一生只需要很少的投资决策。段永平重仓过的公司不到10家。
  2. 董得越深，仓位越重：苹果仓位最高达90%，因为他最懂苹果。集中程度与理解深度成正比。
  3. 分散化稀释收益：投的企业越多，赚的越少。真正的好机会很少，需要重锤出击。
  4. 不做空、不用margin、不做不懂的东西：这三大原则自然导向集中投资，约束了可能的错误范围。

经典案例

  • 苹果：仓位最高达90%，每次下跌都继续买入。这种极度集中建立在对苹果生意模式、企业文化和未来现金流的深度理解之上。
  • 网易、雅虎、苹果、茅台、腾讯：这五个是段永平公开承认的「重手」投资。10年看懂不到10家，重手投5家，约两年一家。
  • 万科：市值100多亿时买入，认为「无论如何也不止」。不需要精确估值，5分钟就能判断便宜。

演变轨迹

  • 早期（2010-2012）：提出投资三大原则，框架上为集中投资奠定基础。
  • 中期（2013-2019）：苹果仓位逐渐加重至90%，用实践验证集中投资理念。
  • 近期（2020-2025）：2025年访谈中明确表示「20个孔的额度还没用完」，核心持仓仍是苹果、茅台、腾讯三只。

常见误解

  • ❌ "集中投资=赌博" → 集中投资建立在深度理解之上，不懂而集中才是赌博。懂而集中是理性。
  • ❌ "集中投资不需要分散风险" → 不是不需要分散，而是真正值得投的公司太少，被迫集中。
  • ❌ "分散投资总比集中好" → 超出一定量后，投的越多赚的越少。过度分散只会摊薄收益。
