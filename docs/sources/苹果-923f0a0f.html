投资时间线

  • 2011年：开始投资苹果（乔布斯去世前后，市场恐慌期）
  • 2012-2013年：持续加仓，深入研究苹果生态闭环和护城河
  • 2013年：苹果开始大规模分红和回购计划（段永平投资逻辑兑现）
  • 2016年：巴菲特开始买入苹果（验证了段永平的判断）
  • 2021年：雪球上明确表示"苹果是非卖品"
  • 2024年：巴菲特股东大会称苹果是比可乐和美国运通"更好的企业"
  • 2024年：段永平开始在高位卖call（持有=买入的纠结）
  • 2025年：方三文对话中确认苹果仍是个人最大仓位

核心要义

  1. 生态闭环 = 超宽护城河：软硬件一体化生态让用户粘性极高——"你要是owner你就明白了，看热闹的怎么说都不会明白的"
  2. "敢为天下后"的典范：苹果不一定是第一个做某产品的，但一定做得最好——"没有创新的天下后肯定死定了"
  3. 现金中性原则：苹果留够运营需要的资金后，多出来的钱全部通过派息或回购返还给股东——这是投资苹果的基本前提
  4. 卖出条件唯一：不是股价高低，而是企业文化是否变质——"为了生意而不是用户体验"
  5. 回购的利好："长期来讲，股价低对苹果没任何坏处，回购到便宜的股票实际上对长期股东也非常好"

经典案例

护城河比可口可乐更宽

段永平对巴菲特说：苹果的护城河比可口可乐宽。理由：可口可乐可以换着喝（百事也很接近），但苹果用户一旦进入生态就很难离开（iMessage、iCloud、App Store、AirPods无缝切换）。巴菲特后来也认同——"苹果是一家更好的企业"。

2013年股价低迷期的逆向思维

2013年苹果股价下滑，市场质疑苹果创新力。段永平反而认为：股价低迷对长期股东有利——苹果用低价回购注销股份，每股内在价值反而在提高。这种"股价越低越开心"的心态，只有真正理解"买股票就是买公司"的人才有。

高位卖call的"持有=买入"实践

2024年当苹果估值较高、短期国债利率5.36%（PE 18.66倍）时，段永平开始卖call——"感觉这个价钱买入确实不便宜了"。这不是看空苹果，而是持有=买入框架下的理性计算：在当前价格继续持有=以当前价格重新买入。

演变轨迹

  • 2011年：乔布斯去世前后开始买入——利用市场恐慌机会
  • 2011-2013年：深度研究苹果商业模式，确认护城河和确定性
  • 2013年：苹果开启回购分红，段永平投资逻辑兑现
  • 2016年：巴菲特买入苹果，两大价值投资者在苹果上达成共识
  • 2019-2025年：持续重仓持有，雪球上频繁为苹果辩护
  • 2024-2025年：开始运用持有=买入框架动态评估苹果仓位

常见误解

误解一："苹果创新力下降，不行了"

段永平认为苹果不需要每次都第一个创新——"敢为天下后"。苹果的核心能力是把已有的东西做到极致，靠生态闭环维系护城河。

误解二："苹果依赖iPhone"

"苹果……以后生态将更强大，更完善。"——苹果的护城河不只是iPhone，而是整个生态（服务收入占比持续提升）。

误解三："苹果估值太高了"

段永平要看的是10年后的角度。他认为在1万亿市值时买入苹果的人已经赚了很多钱——现在看1万亿是极度便宜。
