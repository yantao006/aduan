段永平怎么说

不做空是段永平从巴菲特处学到的三大投资戒律之首。做空风险无限，一次错误就可能致命。长期而言市场向上，做空违背大势。他早年曾犯过做空错误（早年百度），此后坚决不做空。

核心要义

  1. 做空风险无限：做多的最大损失是本金归零，做空的损失理论上无限——股价可以涨到任何价位
  2. 长期大势向上：大市长期一定是向上的，做空违背了基本趋势
  3. 信息不对称：做空一家公司需要了解的东西远比做多要多得多，而且一旦错了就可能是灾难性的
  4. 不做空 ≠ 不用期权：段永平偶尔对了解的股票卖puts和covered calls（风险可控），但绝不做naked calls（无限风险）
  5. 做空的悖论：真正的好机会往往需要有人在大量做空才能有好价钱，但自己做空却是极度危险的

经典案例

早年百度的教训

段永平早年曾经做空百度，结果惨遭损失。这成为他"不做空"原则的亲身教训——"尤其是不要做空长期的好公司，比如早年的百度"。

Naked Calls 的惨痛教训

2010年坦诚分享：以前做过naked calls，错了一次就"一夜差点回到解放前"。此后坚决不再做无限风险的事情。

茅台做空传闻

2013年有媒体误传段永平做空茅台，他当即澄清："很对呀，我不做空，当然不可能做空茅台。其实我是茅台多头哈，已经开始持有茅台了。"

演变轨迹

  • 2010年：首次在博客中明确列为三大原则之一（不做空、不借钱、不做不懂的东西）
  • 2011-2013年：频繁强调并解释不做空的逻辑——"做多市场付给你钱，做空你为市场买单"
  • 2018年：斯坦福演讲中列为Stop Doing List第二条（仅次于不做不懂的东西）
  • 2022年：在雪球上提醒网友"不要把margin当闲钱用，更不要把做空当工具用"
  • 2025年：仍在雪球等平台持续重申不做空原则

常见误解

误解一："做空可以作为对冲工具"

段永平认为，如果你觉得一家公司不好，直接不买就行了，没必要去对冲。做空的复杂度和风险远超想象。

误解二："卖option就是做空"

卖puts和covered calls的风险是有限的（不用margin的前提下），与做空有本质区别。但naked calls=做空，他坚决不做。

误解三："聪明人可以靠做空赚钱"

"芒格知道做空会出大问题，所以就不做空。巴菲特说过，他有很多很厉害的朋友最后都破产了，原因就是因为用了margin。"——真正聪明的人选择不做空。
