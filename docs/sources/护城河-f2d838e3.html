段永平怎么说

"护城河应该是生意模式的一部分。没有护城河的生意模式不是好的生意模式，但有护城河的生意未必就一定是好的生意模式。"
"企业文化是'护城河'的重要部分。很难想象一个没有很强企业文化的企业可以有个很宽的'护城河'。"
"好的生意模式往往具有很宽的护城河。好的生意模式往往是好的未来现金流的保障。"
"'护城河'是用来判断公司内在价值的一个重要手段（不是唯一的）。"
"我喜欢有护城河的生意。我希望拥有一座价值连城的城堡，守护城堡的公爵德才兼备。我希望这座城堡周围有宽广的护城河。"
"我经常对伯克希尔子公司的管理者说，加宽护城河。往护城河里扔鳄鱼、鲨鱼，把竞争对手挡在外面。这要靠服务、靠产品质量、靠成本，有时候要靠专利或营业地点。"
"可口可乐的护城河比30年前更宽了。每次可口可乐在某个国家投资开一家新工厂，它的护城河都会变宽一点。"
"好的商业模式是有护城河的，有定价权的。"

核心要义

  1. 护城河是生意模式的组成部分：护城河从属于好生意模式，但有了护城河也不一定就是好生意。
  2. 企业文化就是护城河：段永平的独到见解——没有强大企业文化的公司不可能拥有宽阔的护城河。
  3. 不断加宽是关键：护城河不是静态的，需要靠服务、产品质量、成本、专利等持续「扔鳄鱼」加固。
  4. 定价权是护城河的核心体现：有护城河的公司往往有定价权，这直接保障了未来现金流。

经典案例

  • 苹果：拥有多层次的护城河——品牌忠诚度、生态锁定、硬件软件一体化、卓越的企业文化。段永平用城堡和公爵的比喻来描述苹果。
  • 茅台：独特的品牌护城河——产地不可复制、文化属性、社交货币属性。定价权极强。
  • Costco：低成本护城河——会员制+极致效率，类似于巴菲特旗下GEICO的护城河逻辑。

演变轨迹

  • 早期（2010-2012）：在《再说对价值投资的一些基本理解》中系统阐述护城河，将护城河与企业文化挂钩是其独特贡献。
  • 中期（2012-2018）：在《啰嗦版》中将护城河纳入生意模式框架。频繁引用巴菲特「城堡与公爵」的比喻。
  • 近期（2019-2026）：在雪球上频繁讨论具体公司的护城河（苹果、茅台、Costco、英伟达等），强调「加宽护城河」是企业经营的核心任务。

常见误解

  • ❌ "有护城河就是好生意" → 有护城河未必就是好生意模式，还需要看是否可持续、是否在加宽。
  • ❌ "护城河=品牌" → 护城河的表现形式很多：低成本（GEICO）、专利、地点、产品质量、服务等。
  • ❌ "护城河是永久性的" → 护城河可能被侵蚀，需要持续加固。连可口可乐都在不断加宽护城河。
