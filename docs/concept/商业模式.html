核心要义

  1. 好商业模式 = 护城河 + 定价权 + 差异化：三者缺一不可。有护城河才有可持续的竞争优势，有定价权才能转移成本，有差异化才能避免价格战
  2. 低毛利是商业模式差的信号：好的赛道天然不会进入低毛利竞争——如果一个行业全行业毛利率都很低，说明这个行业本身商业模式有问题
  3. 商业模式 > 市值：不要被市值大小迷惑，核心看生意本身的好坏。好生意即使市值已经很大，仍然可以持续创造价值
  4. 商业模式是企业文化的另一半：好生意模式（right business）+ 好企业文化（right people）+ 好价格（right price）构成投资决策的铁三角
  5. 商业模式决定未来现金流的确定性：DCF框架下，商业模式越好，未来现金流越可预测、越确定
