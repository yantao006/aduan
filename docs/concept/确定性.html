2010 · 《确定性》

核心要义

  1. 确定性 = 懂的程度：确定性高低的本质是对一家公司"看懂"了多少——懂=能大致判断未来现金流
  2. 护城河 → 确定性 → 安全边际：护城河宽的公司竞争格局稳定，未来预测更可靠，投资确定性更高
  3. 风投 vs 价值投资的分界线：确定性高（苹果）→ 价值投资、重仓；确定性低（拼多多）→ 风投、小仓位
  4. 确定性决定仓位：确定性越高的公司越敢重仓——苹果是最大仓位，拼多多仓位很小
  5. 简化 = 提高确定性：生意模式越简单的公司，未来越容易判断，确定性越高
