    早期（2010-2011）：在网易博客中系统讨论DCF，写有《现金流量折现法》专文，讨论折现率等技术话题。

    中期（2012-2015）：在《投资的信仰》《复习一下》中将DCF升华为「买股票就是买公司」的自然延伸，强调思维框架而非技术工具。

    近期（2019-2025）：坚持DCF框架，明确批驳市盈率决定论。2025年访谈中再次确认「决定投资不是由市盈率来决定的，是由你未来的现金流来决定的」。

原话摘录

未来现金流的折现不是算法，是思维方式，不要企图拿计算器去算出来。当然，拿计算器算一下也没什么错。

2010-05 · 《未来现金流折现》

公司未来现金流的折现就是公司的内在价值。应该在公司股价低于其内在价值时买入。

2010-05 · 《未来现金流折现》

未来现金流折现只是一种思维方式，只有在自己能力圈范围内的公司，投资人才能毛估估看明白。没人真用公式的，至少剩者都是不用公式的。

宁要模糊的精确，也不要精确的模糊。我觉得这就是毛估估的意思。

2010-05 · 《未来现金流折现》

折现率实际上是相对于投资人的机会成本而言的。最低的机会成本就是无风险回报率，比如美国国债的利率。

2010-05 · 《未来现金流折现》

核心要义

  1. 思维方式而非公式：DCF（现金流折现法）是一种思考框架，不是用计算器能精确算出来的算法。
  2. 折现率=机会成本：每个人的折现率应该是自己的长期最大机会成本，对普通人而言最低参考是无风险利率。
  3. 只看净现金流：现金流指净现金流，未来指公司的整个生命周期，不是3年也不是5年。
  4. 能力圈前提：只有在能力圈范围内，投资人才可能毛估估地看明白未来现金流。
