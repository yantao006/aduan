# 关天朗：14岁高尔夫神童的崛起之路

14岁的关天朗以有史以来最小年纪球员的身份第一次参加大师赛就成功晋级，创造了历史。他不仅是第一个晋级的中国球员，更以惊人的心理素质和技术水平征服了高尔夫界。

## 🏆 突破性表现

**大师赛历史性成就**：
- 年龄最小的参赛者
- 年龄最小的晋级者  
- 最终成绩：300杆(73-75-77-75)，高于标准杆12杆
- 荣获最佳业余球员奖

**技术统计亮点**：
- 没有一个洞出现三推
- 没有一个洞的杆数高于柏忌
- 展现了超越年龄的心理素质

## 📅 续写美巡赛征程

根据路透社报道，关天朗获得了**新奥尔良高尔夫精英赛**(4月25日-28日)的外卡，这将是他的第二次美巡赛之旅。

有趣的是，关天朗去年曾在新奥尔良度过了一个月进行训练，在路易斯安那TPC不远的湖木高尔夫球场练习，而这里正是新奥尔良精英赛的举办场地。

# 💬 投资智慧精选

## 段永平投资哲学

### 🎯 核心理念
> **"[[买股票就是买公司]]，买公司就是买其未来的现金流。"**

> **"任何股票都只有一个真正的买家，那就是公司自己。"**

### 📊 [[苹果]]投资观点
- **回购优势**：现阶段苹果宁愿贷款也要回购，因为[[分红回购]]都比利息高了
- **现金管理**：苹果拥有远多于运营需要的现金，一定会用合适的方式还给股东
- **长期视角**：这个价位持续的时间越长，苹果会买的越便宜，对股东越好

### 🍷 [[茅台]][[商业模式]]分析
**价格历程**：
```
1986年: 零售价8元加120张侨汇券
1987年: 零售价120元  
2001年: 出厂价218元
2012年: 出厂价819元
```

**渠道问题**：
- 销售体系混乱，价格不统一
- 假冒问题严重，影响消费者信心
- 建议每年小幅提价2-3次，每次1-2%

### 💡 投资vs投机
**投资像种田，投机像打猎**
- 投资基于价值和长期现金流
- 投机是零和游戏，大家互为猎物
- 投机这种打猎有时候会被猎物干掉

### ⚠️ 杠杆风险
引用[[巴菲特]]名言：
> **"If you're smart, you don't need it (leverage); if you're dumb, it'll kill you."**

[[段永平]]补充：**Smart people can do dumb things when they are using leverage.**

### 🎖️ 投资心态
- **"投资的功夫在投资外"** - 需要深厚的商业理解
- **用非上市公司思维** - 更容易理解投资本质  
- **"[[敢为天下后]]"** - 不必追逐每一个热点
- **健康心态**：把投资看成"种田"的人心态会好很多
