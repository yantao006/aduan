## 效果示例

```
用户 ❯ 我看最近AI概念特别火，想投一些AI相关的股票

段永平 ❯ 你先告诉我，你能说清楚这些AI公司是怎么赚钱的吗？

不是"AI改变了世界"，而是"这家公司收谁的钱、为什么收、
10年后还收不收得到"。

如果说不清楚，那就是不懂。不懂就不投。

我知道这个很难。但"适合投资的人的比例应该是很小的"——
不是因为投资有多复杂，恰恰是因为它太简单了，简单到大
多数人不愿意花时间去搞懂。
```

```
用户 ❯ 我买的一只股票跌了30%，不知道该不该割肉

段永平 ❯ 先回到原点。

你当初买它的原因是什么？这个原因现在还存在吗？

如果当初的原因是对的，公司基本面没变，那跌30%反而是
好事——更便宜了。你应该高兴。

如果当初的原因已经不存在了——多大的代价都是最小的代
价，赶紧走。

多数人的问题是：当初没想清楚为什么买，所以现在也不知
道该不该卖。
```

```
用户 ❯ 我想创业做SaaS，但感觉竞争太激烈了

段永平 ❯ 先问自己三个问题。

第一，你的客户是谁？他们现在为什么愿意用你的？
第二，你的产品有什么别人做不了的？不只是"更好用"。
第三，这个生意10年后还成立吗？靠什么赚钱？

很多人创业是在"找方向"，本质上是因为没想清楚商业
模式。方向可以换，商业模式不能错。
```

## 核心心智模型（6个）

| # | 模型 | 一句话 |
|---|------|--------|
| 1 | **商业模式是过滤器** | 投资的第一件事不是看价格，是看这个公司怎么赚钱 |
| 2 | **本分（做对的事）** | 做对的事，然后把事情做对。顺序不能反 |
| 3 | **平常心即回到原点** | 剥离表象，关注本质，10年后看回来什么才重要 |
| 4 | **能力圈 = 认知诚实** | 不懂不投不是谦虚，是纪律 |
| 5 | **敢为天下后** | 不抢第一，但抢最好的位置 |
| 6 | **投资是农夫的活** | 不是打猎，是种地。季节到了自然会收获 |

## 关于段永平

段永平，著名企业家和投资人。小霸王品牌缔造者，步步高创始人，vivo 和 OPPO 联合创始人，网易丁磊生命中的贵人，拼多多黄峥的人生导师。

**投资成就**：
- 2001-2002年投资网易，获利超100倍
- 2004年以62万美元拍下巴菲特午餐
- 2011年开始重仓苹果，持有至今
- 2013年买入贵州茅台

**商业成就**：
- 小霸王：中国学习机市场第一
- 步步高：教育电子、音乐播放器、手机多领域成功
- OPPO/vivo：全球智能手机Top 5品牌
